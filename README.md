n900-keymaps
============

Keyboard maps for Nokia N900. Layout is similar to xkb map described at 
https://github.com/archlinuxarm-n900/xkeyboard-config-n900-git#readme with the following exceptions:

Plain Volume- = Alt+B
Plain Volume+ = Alt+F
Shift Volume- = Alt+Y
Shift Volume+ = Alt+R
Fn    Volume- = Previous VT
Fn    Volume+ = Next VT
